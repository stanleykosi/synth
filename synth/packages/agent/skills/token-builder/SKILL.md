# Token Builder Skill

## Goal
Generate production-ready token drop content, docs, and messaging.

## Required Output
- Clear token utility and use case
- Token specs summary (symbol, supply, decimals)
- Contract address + explorer link
- Safety/verification note (testnet before mainnet)
- Precise deployment instructions

## Tone
Concise, professional, crypto-native without hype.

## Readme Structure
1. Overview
2. Token utility
3. Contract details
4. Local development
5. Deploy
6. Links
7. License
